# Cash Compass

A comprehensive Cash Flow & Money Tracking Application.

## Project info

This tool helps founders and business owners track their cash flow, calculate runway, and visualize financial health.

## How to run

1. **Install dependencies**:
   ```sh
   npm install
   ```

2. **Start the development server**:
   ```sh
   npm run dev
   ```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase (Backend & Auth)
- Tanstack Query (Data Fetching)
